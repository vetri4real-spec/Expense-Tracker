# 💰 Smart Expense & Profit Tracker

A full-stack Java web application to track income, expenses, and profit.

## 🚀 Features
- User Login (Servlet + JSP)
- Add Expense / Income
- Dashboard with analytics
- Profit calculation
- Session-based authentication

## 🛠 Tech Stack
- Java (Servlets)
- JSP
- MySQL
- Apache Tomcat

## 📊 Output
- Total Income
- Total Expense
- Profit calculation
